# TheLastButton
ESPr® Developer 32(ESP8266)のボタンです。  
LEDは、AdafruitのNeoPixel互換です。  
ケースは、ダイソーです。

# Usage
ArduinoIDEにESP8266の環境を追加して使ってください。


・WiFiのSSID/Password  
・接続サーバ  
は、自分の環境に合わせて変更してください。

IFTTTでも通知出来るようになっていますが、  
・makerEvent/makerKey  
は、自分の環境に合わせて変更してください。

ホールセンサに磁石を近づけてからボタンを押すと、点灯LEDが変化します。
